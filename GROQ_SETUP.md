# Getting Your Free Groq API Key

## Quick Steps

1. **Go to Groq Console**: https://console.groq.com/keys

2. **Sign Up / Log In**:
   - Use your Google, GitHub, or email account
   - No credit card required!

3. **Create API Key**:
   - Click "Create API Key"
   - Give it a name (e.g., "Event-Driven Chat App")
   - Copy the key (starts with `gsk_`)

4. **Use in the App**:
   - Start the dev server: `npm run dev`
   - Open http://localhost:3000
   - Paste your API key when prompted
   - Start chatting!

## Groq Free Tier Limits

- **30 requests per minute**
- **14,400 requests per day**
- **No credit card required**
- **Fast inference** using Groq's LPU technology

## Troubleshooting

### "Invalid API key" error
- Make sure you copied the entire key (starts with `gsk_`)
- Check for extra spaces before/after the key
- Generate a new key if needed

### Rate limit exceeded
- Free tier: 30 requests/minute
- Wait a minute and try again
- Or upgrade to a paid plan for higher limits

### API not responding
- Check Groq status: https://status.groq.com/
- Verify your internet connection
- Try regenerating your API key
